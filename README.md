# minikube-yaml

```
minikube config set driver docker
minikube start
```

```
minikube status
```

start dashboard
```
minikube dashboard
```

install ingress
```
minikube addons enable ingress
```



show nginx-app-service url
```
minikube service nginx-app-service --url
```

stop
```
minikube stop
```

delete clean
```
minikube delete
minikube delete --all
minikube delete --purge
```
